# Cafe Shop Project  

## Project Overview  
This project is a simple web application for a cafe shop. It includes a homepage (`cafe_shop.html`) that displays the cafe's menu, images, and some interactive elements using JavaScript. The design is styled with CSS for a clean and modern look.  

## File Structure  
- **`cafe_shop.html`**: The main HTML file that serves as the homepage.  
- **`CSS/style.css`**: Stylesheet for the project, containing all the CSS rules.  
- **`JS/script.js`**: JavaScript file for interactive elements.  
- **`IMG/`**: Directory containing images used in the project (e.g., logos, food photos).  

## How to Run  
1. Open the `cafe_shop.html` file in a web browser.  
2. The page will load the CSS and JavaScript files automatically.  
3. No additional setup is required; the project is ready to use.  

## How the Project Works  

This project is a simple web application that uses HTML, CSS, and JavaScript to create an interactive cafe website. The homepage (`cafe_shop.html`) is the main entry point, styled with CSS for layout and appearance, and enhanced with JavaScript (`JS/script.js`) for interactive features like menu filtering, reservation form handling, dark mode toggling, and smooth scrolling. Images are stored in the `IMG/` directory and referenced in the HTML and CSS. The site is designed to be opened directly in a web browser without any server setup.
